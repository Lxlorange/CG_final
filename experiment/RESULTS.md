# 当前实验结果

运行命令：

```powershell
python experiment\run_experiment.py
```

当前平均指标：

| 方法 | PSNR | SSIM | 平均渲染时间 |
| --- | ---: | ---: | ---: |
| traditional_point_splat | 19.87 | 0.706 | 0.834 s |
| neural_ray_field | 10.52 | 0.271 | 0.345 s |

结果解释：

传统点云 splatting 的质量明显高于轻量坐标场，原因是它显式利用了训练视角深度和三维反投影，能够保留基本几何、遮挡和棋盘纹理。然而它仍然存在点状噪声、边界破碎和未观测区域缺失。

轻量坐标场的结果明显模糊，说明仅将相机射线、像素坐标和视角角度输入到一个回归模型，并不能可靠学习三维场景结构。这一结果不能代表 NeRF 或 3D Gaussian Splatting 的真实性能，但可以作为一个反例：神经渲染的关键不是泛泛地使用学习模型，而是要把学习表示嵌入三维可见性和渲染方程中。

论文正式实验应继续补充真正 NeRF 或 3DGS 方法的输出，并与当前传统 baseline 放在同一测试视角下比较。
